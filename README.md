# 💼 Employee Salary Data Analysis

## 📌 Project Description
This project focuses on analyzing employee salary data using Python. It helps in understanding salary distribution, identifying high-paid employees, and comparing salaries across departments.

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Features
- Data cleaning and preprocessing  
- Salary distribution analysis  
- Department-wise salary comparison  
- Identification of highest and lowest paid employees  
- Data visualization using graphs  

---

## 📁 Project Files
- `employee_salary.py` → Main Python script  
- `employee_data.csv` → Dataset used for analysis
